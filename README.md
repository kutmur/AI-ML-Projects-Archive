## İçerik Yapısı

Projeler, makine öğrenmesi ana başlıkları altında kategorize edilmiştir:

### Supervised Learning
- Regresyon algoritmaları (Linear, Polynomial, Ridge, Lasso)
- Sınıflandırma algoritmaları (Logistic Regression, SVM, Decision Trees)
- Ensemble yöntemleri (Random Forest, XGBoost, LightGBM)

### Unsupervised Learning
- Kümeleme (K-Means, DBSCAN, Hierarchical)
- Boyut indirgeme (PCA, t-SNE, UMAP)
- Anomali tespiti

### Deep Learning
- Yapay Sinir Ağları (ANN)
- Konvolüsyonel Sinir Ağları (CNN)
- Tekrarlayan Sinir Ağları (RNN, LSTM, GRU)
- Transfer Learning ve Fine-tuning

### Natural Language Processing (NLP)
- Metin işleme ve temizleme
- Sentiment Analysis
- Named Entity Recognition (NER)
- LLM (Large Language Models) denemeleri
- Chatbot projeleri

### Computer Vision
- Görüntü işleme
- Nesne tespiti (Object Detection)
- Görüntü sınıflandırma
- Image Segmentation
- Yüz tanıma

### Reinforcement Learning
- Q-Learning
- Deep Q-Networks (DQN)
- Policy Gradients
- Oyun AI'ları

## Nasıl Katkı Sağlarım?

Projeni eklemek için **[CONTRIBUTING.md](CONTRIBUTING.md)** dosyasını incele!

### Hızlı Başlangıç

```bash
# 1. Repository'yi fork et
# 2. Clone et
git clone https://github.com/KULLANICI_ADIN/AI-ML-Projects-Archive.git
cd AI-ML-Projects-Archive

# 3. Yeni bir branch oluştur
git checkout -b proje-ekleme-ismin

# 4. Projenizi ekleyin ve commit edin
git add .
git commit -m "[Kategori] Proje İsmi - İsim eklendi"

# 5. Push edin ve PR oluşturun
git push origin proje-ekleme-ismin
```

## Kurallar

### Yapılması Gerekenler
- Kodlarınızda **açıklayıcı yorumlar** (comments) bulundurun
- Her proje için **README.md** dosyası ekleyin
- **Temiz ve anlaşılır** kod yazın
- **Sadece** `.py`, `.ipynb` ve gerekli küçük dosyaları yükleyin
- Görselleştirmeler ve sonuçlar ekleyin

### Yapılmaması Gerekenler
- Model ağırlıkları (weights) yüklemeyin → **Google Drive/Hugging Face linki** verin
- Büyük veri setleri yüklemeyin → **Kaggle/UCI linki** paylaşın
- Gereksiz dependency'ler eklemeyin
- Başkalarının kodunu izinsiz kopyalamayın

### Dosya Boyutu Limitleri
- **Tek dosya:** Max 10 MB
- **Veri setleri:** Max 5 MB (daha büyükleri için link verin)
- **Model dosyaları:** Link ile paylaşın
