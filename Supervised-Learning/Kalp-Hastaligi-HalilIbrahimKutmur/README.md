# Heart-Disease-Prediction-HalilIbrahimKutmur

## Hakkında
Bu proje, makine öğrenmesi (Logistic Regression) kullanarak hastaların sağlık verilerine göre kalp hastalığı riskini tahmin eder. Model, yaş, cinsiyet, kolesterol seviyesi gibi 13 farklı klinik özelliği analiz ederek ikili sınıflandırma (var/yok) yapar.

## Veri Seti
- **Kaynak:** UCI Machine Learning Repository (Heart Disease Dataset)
- **Boyut:** 303 satır, 13 özellik (Age, Sex, CP, Trestbps, Chol, Fbs, Restecg, Thalach, Exang, Oldpeak, Slope, Ca, Thal)
